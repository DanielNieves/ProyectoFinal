Descripción del Proyecto

El proyecto consiste en desarrollar un dispositivo portátil basado en Arduino que mida la temperatura corporal y el ritmo cardíaco de los pacientes. Los datos se mostrarán en una pantalla LCD y se almacenarán en una base de datos MySQL para su monitoreo y análisis posterior. Además, el sistema enviará alertas a un servidor web si se detectan valores anormales.
